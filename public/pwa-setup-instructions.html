⏳ Íconos PNG Pendientes

Necesitas generar los 8 tamaños PNG para completar la PWA

⚠️ Paso Final Requerido

La PWA está técnicamente lista, pero necesita los íconos PNG en todos los tamaños para que los usuarios puedan instalarla en sus smartphones como una app nativa.

📝 Instrucciones Paso a Paso

1
Generar Íconos PNG

Tienes 3 opciones para generar los íconos en todos los tamaños necesarios:

Opción A: Generador Integrado (Más Fácil)

🎨 Abrir Generador de Íconos

Sube una de tus imágenes del logo oficial, se generarán automáticamente todos los tamaños, y descárgalos con un clic.

Opción B: RealFaviconGenerator (Online)

🌐 Ir a RealFaviconGenerator

Sube la imagen más grande que tengas (512x512 preferiblemente) y descarga el paquete completo.

Opción C: Manualmente con Canva/Photoshop

Exporta la imagen en cada uno de estos tamaños: 72, 96, 128, 144, 152, 192, 384, 512

2
Renombrar Archivos

Los archivos PNG deben tener exactamente estos nombres:

  • icon-72x72.png
  • icon-96x96.png
  • icon-128x128.png
  • icon-144x144.png
  • icon-152x152.png
  • icon-192x192.png ⭐ OBLIGATORIO
  • icon-384x384.png
  • icon-512x512.png ⭐ OBLIGATORIO

Los tamaños 192x192 y 512x512 son obligatorios para que la PWA funcione.

3
Guardar en el Proyecto

Coloca todos los archivos PNG en la carpeta:

/public/icons/

Puedes arrastrar y soltar los archivos directamente en tu editor de código (VS Code, etc.)

4
Hacer Commit y Push

Una vez que todos los archivos PNG estén en la carpeta, sube los cambios a Git:

Opción A: Terminal

git add .
git commit -m "🎨 Agregar íconos PWA oficiales"
git push

Opción B: VS Code

1. Abre el panel "Source Control" (Ctrl+Shift+G)
2. Haz clic en el "+" para agregar los cambios
3. Escribe el mensaje: 🎨 Agregar íconos PWA oficiales
4. Haz clic en "Commit" y luego "Sync Changes"

Opción C: GitHub Desktop

1. Abre GitHub Desktop
2. Verás los archivos nuevos en la lista
3. Escribe el mensaje de commit
4. Haz clic en "Commit to main" y luego "Push origin"

5
Verificar en Vercel

Después del push, Vercel desplegará automáticamente los cambios:

✅ El deployment toma 1-2 minutos
✅ Recibirás una notificación cuando termine
✅ Los íconos estarán disponibles en producción

🚀 Ver App en Producción
6
Probar la Instalación

Una vez desplegado, prueba instalar la PWA:

En Android (Chrome):

1. Abre la app en Chrome
2. Toca el menú (⋮) → "Instalar app" o "Agregar a pantalla de inicio"
3. Verás el ícono oficial con la "U" roja
4. ¡Listo! Ya tienes la app instalada

En iPhone (Safari):

1. Abre la app en Safari
2. Toca el botón de compartir (□↑)
3. Selecciona "Agregar a inicio"
4. El ícono aparecerá en tu pantalla de inicio

En Desktop:

1. Abre la app en Chrome/Edge
2. Verás un ícono de instalación (⊕) en la barra de direcciones
3. Haz clic para instalar
4. La app se abrirá en su propia ventana
