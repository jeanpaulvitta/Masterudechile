# 🚀 INICIO RÁPIDO - Completar Íconos PWA

## 👋 ¡Hola!

Has subido **4 imágenes del logo oficial** de Master UCH (con la "U" roja sobre fondo azul).

Ahora solo necesitas **5 minutos** para completar la PWA y hacerla instalable en smartphones.

---

## ⚡ ACCIÓN INMEDIATA (3 pasos):

### 1️⃣ Abrir el Generador (1 click)

**Opción A - Local (si estás desarrollando):**
```
http://localhost:5173/pwa-tools.html
```

**Opción B - Producción (después del deploy):**
```
https://app-masteruchile.vercel.app/pwa-tools.html
```

### 2️⃣ Generar los Íconos (2 minutos)

1. Haz clic en **"Generador de Íconos"**
2. Sube **UNA** de las 4 imágenes que compartiste (la más grande)
3. Espera a que se generen los 8 tamaños
4. Haz clic en **"Descargar Todos los Íconos"**
5. Se descargarán 8 archivos PNG

### 3️⃣ Integrar al Proyecto (3 minutos)

**En tu computadora:**

1. Abre tu proyecto en VS Code (o tu editor favorito)

2. Ve a la carpeta:
   ```
   /public/icons/
   ```

3. Arrastra y suelta los 8 archivos PNG que descargaste

4. Abre la terminal y ejecuta:
   ```bash
   git add .
   git commit -m "🎨 Agregar íconos PWA oficiales Master UCH"
   git push
   ```

5. **¡Listo!** Vercel desplegará automáticamente en 1-2 minutos

## 📱 Probar la PWA:

### En tu smartphone Android:
1. Abre Chrome
2. Ve a: https://app-masteruchile.vercel.app/
3. Menú (⋮) → **"Instalar app"**
4. ¡Verás el ícono con la "U" roja en tu pantalla de inicio!

### En iPhone:
1. Abre Safari
2. Ve a: https://app-masteruchile.vercel.app/
3. Botón compartir (□↑) → **"Agregar a inicio"**
4. ¡App instalada!
