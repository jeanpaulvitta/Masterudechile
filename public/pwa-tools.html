📋 ¿Qué necesitas hacer?

Has subido 4 imágenes del logo oficial de Master UCH. Ahora necesitas generar los íconos PNG en 8 tamaños específicos para completar la PWA.

  1. Generar íconos - Usa el generador para crear los 8 archivos PNG
  2. Verificar - Comprueba que todos los íconos estén correctos
  3. Guardar - Coloca los archivos en /public/icons/
  4. Deploy - Haz commit y push para publicar
  5. ¡Listo! - Tu PWA estará instalable en smartphones

⚡ Flujo Recomendado

  1. Genera los íconos
    → Haz clic en "Generador de Íconos"
    → Sube una de tus 4 imágenes del logo
    → Descarga los 8 archivos PNG generados
  2. Guarda en el proyecto
    → Abre tu proyecto en VS Code (o tu editor)
    → Ve a la carpeta /public/icons/
    → Arrastra los 8 archivos PNG
  3. Verifica los íconos
    → Usa el "Verificador de Íconos"
    → Asegúrate que todos estén correctos
  4. Sube los cambios
    → Haz commit: git commit -m "🎨 Agregar íconos PWA"
    → Push a GitHub: git push
  5. ¡Prueba la PWA!
    → Espera 1-2 minutos al deployment
    → Abre la app en tu smartphone
    → Instálala en tu pantalla de inicio
